## Stopwatch

A simple stopwatch built with React and Electron.

### To run locally:

- Clone repo: 
- run `npm install`
- run `npm run dev`

### To package:

#### Windows

- run `npm run package-win`

#### Mac

- run `npm run package-mac`

#### Linux

- run `npm package-linux`
